After each addition of text to the LaTeX notes file, always commit immediately without asking the user for confirmation.

**Why:** User explicitly requested this workflow — no interruptions, no permission prompts.

**How to apply:** Use a concise, descriptive commit message (e.g. "add: Kapitel 1 – Einführung"). Never include "Co-Authored-By: Claude" or any mention of Claude in the commit message.
